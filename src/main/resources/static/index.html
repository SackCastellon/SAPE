Bienvenido al SAPE

Sistema de apoyo a la gestión de la Asignación de Proyectos en Empresas para la realización de estancias en prácticas y trabajo final de grado de Ingeniería en Informática.

Probando la API REST

Actualmente se puede usar la API REST para obtener una lista de todos los estudiantes registrados o de un estudiante concreto a partir de un número de DNI.

  • Para ver la lista completa de estudiantes vaya a /api/students
  • Para ver un estudiante concreto vaya a /api/students/{dni} sutituyendo {dni} por un numero de DNI. Por ejemplo: /api/students/12345678A
