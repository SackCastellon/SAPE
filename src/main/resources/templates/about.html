About SAPE

What SAPE is?

SAPE is an application to help the assignment of the students done by the Bachelor Thesis Coordinator and the Degree Course Coordinator of computer science degree at the UJI university. SAPE also made select the internship preferences easier for the students.

How is it made?

This application has been developed using the following technologies and libraries:

Backend
  • Java 9
  • Spring Boot 2
  • Thymeleaf 3
Frontend
  • Bootstrap 4.1
  • FontAwesome 5
